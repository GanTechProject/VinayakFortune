# VentureMiner AI — Documentation Suite

> **Project:** VentureMiner AI — AI Venture Intelligence Platform
> **Suite version:** v1.0
> **Last updated:** 2026-07-20
> **Total documents:** 38 (Documents 00–37)

This directory contains the full documentation suite for VentureMiner AI, generated from the project requirements discussed in `ChatHistory.txt`. Every document exists in two forms:

1. **Source Markdown** (`.md`) in its primary folder — editable, diffable, source of truth.
2. **Generated PDF** (`.pdf`) in `docs/pdfs/` — formatted, cover page, TOC, page numbers, ready to share.

## How to use this suite

- **First read:** Document 00 (Governance) → Document 01 (PRD) → Document 02 (TRD).
- **Engineers:** start with Document 05 (Backend), then Document 07 (System Architecture), then your specialty (08–18 for AI, 19–30 for engineering).
- **Product / design:** Documents 01, 03, 04.
- **Security / compliance:** Documents 21, 29.
- **Operations / SRE:** Documents 18, 23, 25, 28.
- **Anyone writing code:** Document 26 (Coding Standards), and for the web app Document 27.

## Regenerating PDFs

From this directory:

```
python generate_pdfs.py
```

Options:

- `python generate_pdfs.py --only 01-PRD/01_product_requirements_document.md` — single document.
- `python generate_pdfs.py --skip-existing` — incremental.

Dependencies: `reportlab>=5.0`, `markdown>=3.5` (both already installed).
